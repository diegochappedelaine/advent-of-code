# Advent of code 2022

Solutions for [Advent of code 2022](https://adventofcode.com/2022) in TypeScript.

## Getting started

This project is using [Deno](https://deno.land/)

## Running the code

```bash
$> yarn day-1
$> yarn day-2
...
```
